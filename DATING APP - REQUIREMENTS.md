# **Requirements**

| Module Name | Details |
| ----- | ----- |
| **0\. Landing Page** | Hero Banner About the App Footer |
| **1\. User Registration & Login** | New User Sign‑Up (Web App) Register using email Enter name, age, short bio (text), and upload a profile picture |
|  | Returning User Login Secure login with email and password |
| **2\. User Profile Management** | View your profile in the browser |
|  | Edit details: name, bio, profile photo |
| **3\. User Discovery & Matching** | Browse profiles via desktop interface |
|  | Swipe (drag) right to like, left to skip |
|  | Form a match when both users like each other |
|  | Avoid showing the same profile again |
|  | Filters: age, distance |
| **4\. Messaging / Chat** | Chat unlocked only after matching |
|  | Send and receive text messages |
| **5\. Match List** | Display all current matches |
|  | Allow users to unmatch (removes chat access) |
| **6\. Bonus Features** | Browser‑based push notification simulation for new matches/messages |
|  | Light/dark mode UI toggle |
